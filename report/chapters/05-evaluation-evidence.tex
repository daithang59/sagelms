\section{Phương pháp đánh giá}

Đồ án được đánh giá theo hai lớp: lớp ứng dụng và lớp pipeline. Lớp ứng dụng chứng minh SageLMS có thể build, test và chạy các service chính. Lớp pipeline chứng minh quy trình DevSecOps có thể kiểm soát thay đổi từ pull request đến runtime trên GKE. Khác với Chương~\ref{chap:implementation}, chương này chỉ tập trung vào câu hỏi: mỗi luận điểm quan trọng của báo cáo được chứng minh bằng evidence nào.

Các nhóm evidence chính gồm:

\begin{itemize}
  \item GitHub Actions workflow logs.
  \item Trivy image scan reports.
  \item CycloneDX SBOM files.
  \item Image digest reports.
  \item Deploy summary và deploy PR body.
  \item OpenTofu/Checkov validation evidence.
  \item GKE, CloudNativePG, Redis và ESO status.
  \item Post-deploy rollout và smoke test artifacts.
\end{itemize}
