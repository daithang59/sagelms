\chapter{API contracts}
\label{appendix:api-contracts}

Các OpenAPI contracts hiện có trong thư mục \path{contracts/openapi}. Đây là nền để sinh API reference, mock hoặc contract testing trong các bước mở rộng \cite{openapiSpec}. Tuy nhiên, phụ lục này chỉ dùng để công khai trạng thái contract, không dùng để thay thế evidence runtime ở Chương~\ref{chap:evaluation}. Cần phân biệt contract đã khớp implementation với contract còn mang tính bản nháp: AI Tutor hiện đã chạy theo implementation mới, nhưng file contract tương ứng vẫn cần đồng bộ lại trước khi dùng làm nguồn sinh client hoặc contract test.

Với AI Tutor, implementation hiện có các endpoint chính: \path{/health}, \path{/api/v1/ai/chat}, \path{/api/v1/ai/conversations}, \texttt{/api/v1/ai/conversations/\{id\}/messages}, \texttt{/api/v1/ai/conversations/\{id\}} cho đổi tên và xóa mềm hội thoại. Contract cũ còn mô tả hướng RAG ingestion qua \path{/api/v1/ai/ingest} và hỏi đáp qua \path{/api/v1/ai/ask}; phần này nên được xem là backlog đồng bộ contract, không phải trạng thái runtime hiện tại.
