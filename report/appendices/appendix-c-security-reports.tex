\section{SBOM}

SBOM CycloneDX được lưu dưới \path{reports/sbom/<git-sha>}. Mỗi service có một file \path{*-sbom.cdx.json}. Báo cáo chỉ trích metadata, đường dẫn artifact và ý nghĩa truy vết thành phần phần mềm; không đưa toàn bộ SBOM vào phụ lục vì nội dung dài.

\placeholderfigure{Placeholder ảnh SBOM CycloneDX}{Vị trí chèn ảnh metadata đầu file SBOM hoặc artifact list dưới \path{reports/sbom/<git-sha>}, thể hiện tên service, định dạng CycloneDX và commit tương ứng.}

\section{Cosign signature và SBOM attestation}

CD workflow lưu log Cosign trong artifact \path{reports/cosign/*.txt}. Với run \texttt{26303390987}, auth-service commit \texttt{a8a0b63d1568f0c356d10a22fcb89abad429ef52} đã pass cả \texttt{cosign verify} và \texttt{cosign verify-attestation} trên digest \path{sha256:9e436d64f9a39a65d8052c57d9da5abbf285f5b400f4c94610311fe543efa35d}.
